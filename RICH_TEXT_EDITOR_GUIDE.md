## Overview
Rich Text Editor telah ditambahkan ke halaman Kelola Berita, memungkinkan Anda membuat konten berita yang lebih kaya dan menarik dengan format teks yang beragam.

## Fitur-Fitur yang Tersedia

### 1. **Format Teks Dasar**
- **Bold (Tebal)** - `Ctrl/Cmd + B`
- **Italic (Miring)** - `Ctrl/Cmd + I`
- **Underline (Garis Bawah)** - `Ctrl/Cmd + U`
- **Strike (Coret)** - Format strikethrough untuk teks

### 2. **Heading (Judul)**
- Heading 1-6 tersedia
- Gunakan untuk struktur konten yang jelas
- H1: Judul utama
- H2: Subjudul
- H3-H6: Sub-sub judul

### 3. **Lists (Daftar)**
- **Ordered List (Bernomor)** - Untuk langkah-langkah atau urutan
- **Bullet List (Titik)** - Untuk daftar item

### 4. **Alignment (Perataan)**
- Left Align (Rata Kiri)
- Center Align (Rata Tengah)
- Right Align (Rata Kanan)
- Justify (Rata Kanan-Kiri)

### 5. **Indentation (Indentasi)**
- Indent (Masuk)
- Outdent (Keluar)
- Berguna untuk membuat struktur bertingkat

### 6. **Warna**
- **Text Color** - Ubah warna teks
- **Background Color** - Highlight dengan warna latar

### 7. **Blockquote (Kutipan)**
- Untuk menampilkan kutipan atau highlight teks penting
- Ditampilkan dengan border emas di sebelah kiri

### 8. **Code Block**
- Untuk menampilkan kode program
- Background gelap dengan syntax highlighting

### 9. **Media**
- **Link** - Tambahkan hyperlink ke teks
- **Image** - Sisipkan gambar dari URL
- **Video** - Embed video dari URL

### 10. **Font & Size**
- Pilihan berbagai font
- Ukuran: Small, Normal, Large, Huge

### 11. **Script**
- Subscript (H‚ÇÇO)
- Superscript (x¬≤)

### 12. **Clean Format**
- Tombol "Clean" untuk menghapus semua format

## Tips Penggunaan

### ‚úÖ Best Practices

1. **Gunakan Heading untuk Struktur**
   - Mulai dengan H1 untuk judul utama
   - H2 untuk bagian-bagian utama
   - H3-H6 untuk sub-bagian

2. **Paragraf yang Jelas**
   - Pisahkan paragraf dengan enter
   - Jangan terlalu panjang (3-5 kalimat per paragraf)

3. **Gunakan Lists**
   - Memudahkan pembaca memahami poin-poin penting
   - Lebih terstruktur daripada paragraf panjang

4. **Blockquote untuk Emphasis**
   - Gunakan untuk kutipan penting
   - Atau untuk highlight informasi kunci

5. **Link yang Jelas**
   - Berikan teks link yang deskriptif
   - Hindari "klik di sini"

6. **Gambar dengan Caption**
   - Tambahkan deskripsi di bawah gambar
   - Gunakan alt text yang bermakna

   - Hindari bold + italic + underline sekaligus

## Shortcut Keyboard

- **Bold**: `Ctrl/Cmd + B`
- **Italic**: `Ctrl/Cmd + I`
- **Underline**: `Ctrl/Cmd + U`
- **Link**: `Ctrl/Cmd + K`
- **Undo**: `Ctrl/Cmd + Z`
- **Redo**: `Ctrl/Cmd + Y` atau `Ctrl/Cmd + Shift + Z`

## Troubleshooting

### Format Tidak Muncul di Website?
- Pastikan konten sudah disimpan
- Refresh halaman detail berita
- Clear cache browser

### Gambar Tidak Muncul?
- Pastikan URL gambar valid dan accessible
- Gunakan HTTPS untuk URL gambar
- Cek ukuran file tidak terlalu besar

### Video Tidak Embed?
- Gunakan URL embed, bukan URL halaman
- YouTube: Gunakan URL embed dari "Share > Embed"
- Format: `https://www.youtube.com/embed/VIDEO_ID`

## Support

Jika mengalami masalah dengan Rich Text Editor:
1. Clear browser cache
2. Coba di browser berbeda
3. Pastikan koneksi internet stabil
4. Kontak admin sistem
